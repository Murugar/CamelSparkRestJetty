Rest Service Feeder

Camel, Spark REST by Apache

This example shows how to use Spark REST to define REST endpoints in Camel routes using the Java DSL

There is a user REST service that supports the following operations

  • view/id - to view a user with the given id
  • list - to view all users
  • update - to update/create an user
The view and list operations are HTTP GET, and update is using HTTP PUT. From a web browser you can access the first two services using the following links
  • user/view/123 - to view the user with id 123
  • user/list - to list all users
